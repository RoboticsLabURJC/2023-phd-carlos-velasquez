Debido a que el enfoque de detección de carriles implementado utilizando visión computacional clásica no demostró la robustez deseada, se optó por generar un dataset de manera manual. Este proceso involucró la etiquetación manual de un conjunto de datos utilizando la herramienta Labelme. Se crearon etiquetas para tres clases principales: left_lane (carril izquierdo), right_lane (carril derecho) y background (fondo).
